NeuralBlock

Welcome!

This is the project page for NeuralBlock (NB), a neural network built using Keras/Tensorflow that detects in-video YouTube sponsorships. You can find the project on Github here. Additionally, this web application provides an API to SponsorBlock, whom supplied crowd-sourced sponsor labels that allowed NB to be trained.

Demo

Submit a video ID and see where the sponsorships are. For true predictions, submit videos published after March 3, 2020.
Example: https://www.youtube.com/watch?v=QqEuO5im1nA
